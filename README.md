# Production Monitoring Dashboard

Data collection and visualization workflow developed using Excel and Power BI to support operational decision-making in a manufacturing environment.

## Overview
This project demonstrates a production monitoring solution where operational data is structured and transformed into visual insights through Power BI dashboards.

## Features
- KPI monitoring
- Production tracking
- Trend analysis
- Data visualization
- Operational insights

## Technologies
- Power BI
- Excel
- Power Query
- Data Modeling
